# Audio Param Graph

Graphs the function of a series of [web audio params](https://developer.mozilla.org/en-US/docs/Web/API/AudioParam).

Hosted [hereh](http://briandunn.github.io/audio_param_graph)

## Dev

    $ npm install
    $ node run
    $ open http://localhost:1337
